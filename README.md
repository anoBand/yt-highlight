# yt-highlight
 
 유튜브 영상에서 하이라이트 클립을 추출

# 개발 환경에서 실행 하는법

1. node.js 설치
2. 폴더를 생성 후 IDE로 열기
3. 터미널에서 git clone https://github.com/fullsack73/yt-highlight .
4. 터미널에서 npm install
5. 터미널에서 npm run dev
